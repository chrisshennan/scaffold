# Scaffold

## Usage

```
# Clone the repo
git clone --depth=1 https://github.com/chrisshennan/scaffold.git

# Reset the origin 
cd scaffold
git remote remove origin
git remote add origin [new origin]
git push origin main

# Start the project and initialise it.
make start
make initialize
```
